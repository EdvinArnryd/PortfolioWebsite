Introduction

Finite State Machine Drone AI built in Unity. The goal of this project was to create a drone that drops bombs in random locations. Using a finite statemachine, The drone automatically controls where to go, when to perform different actions and return to the start of the loop. Using the animation controller to sync the statemachine with the animation states, made the drone come to life.

For the Bombs I used object pooling for better performance and clean up. Using Unity's own pooling system made it easy to add the necessary functions and let Unity handle the rest.
